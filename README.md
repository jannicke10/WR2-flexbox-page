# Nils Bjarte Memorial Page
A small two-page website made for Oslo Nye Fagskole.

## Project Purpose

This project demonstrates:

- Semantic HTML structure
- Two different Flexbox layouts
- A basic HTML5 form with validation
- Git/GitHub workflow with meaningful commits

## Pages and Files

- `html.html`  
  Main page with story content, fun facts, and contact form. Aligned flexbox layout included.
- `photos.html`  
  Gallery page with photo cards and captions. Centered flexbox layout included.
- `stylesheet.css`  
  Shared styling for both pages.
- `.gitignore`  
  Ignores local/system files.
- `LICENSE`  
  Project license file.

## Assignment Requirement Coverage

- Semantic tags used: `header`, `nav`, `main`, `section`, `footer`
- Main heading and at least three sections included
- Flexbox used in:
  - navigation (`.main-nav`)
  - fun facts layout (`.flex-container`)
  - centered gallery layout (`.photo-grid`)
- Form contains:
  - name (`text`)
  - email (`email`)
  - message (`textarea`)
- Form uses labels and HTML5 validation (`required`, `type="email"`)
- Basic responsive behavior with wrapping and media queries

## How to Run

1. Clone the repository.
2. Open `html.html` in your browser.
3. Use the navigation to open the gallery (`photos.html`).
